# OpenClaw Lighthouse 🦞

A lightweight public workbench for Christopher and OpenClaw.

This repo is meant to hold readable artifacts, project context, and simple web pages that can be viewed from Telegram or any browser. It is deliberately optimized for smaller/free-tier models: clear files, short instructions, and low-token context.

- Landing page: `index.html`
- Operating context: `context/`
- Generated/reference artifacts: `artifacts/`
